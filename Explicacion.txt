En este proyecto se integraron electrónica, robótica y ciencia de datos mediante el desarrollo de un sistema de monitoreo ambiental con Arduino. Se utilizaron sensores de luz (LDR), gas (MQ2) y humedad para recolectar datos en distintos períodos del día (mediodía, noche con luz y noche sin luz). Los datos se transmitieron por USB y se guardaron en un archivo CSV mediante un script en Python que utiliza PySerial. Posteriormente, se analizaron y visualizaron los datos en Jupyter Notebook con herramientas como pandas, matplotlib y seaborn, generando histogramas, mapas de calor y boxplots para comparar las condiciones ambientales. Finalmente, se implementó un modelo de clasificación usando RandomForestClassifier para predecir el período del día a partir de las mediciones, obteniendo excelentes resultados, lo que demuestra la exitosa integración de hardware y software en un sistema inteligente de monitoreo ambiental.
